📝 ToDo List Application

A full-stack ToDo List application built using Angular and .NET Core Web API.
The application includes user login functionality and allows users to manage daily tasks efficiently.

🚀 Features
🔐 User Login Authentication
➕ Add new tasks
✏️ Update tasks
❌ Delete tasks
📋 View task list

🛠️ Technologies Used
Frontend
Angular
TypeScript
HTML5
CSS3
Bootstrap

Backend
.NET Core Web API
C#
SQL Server
